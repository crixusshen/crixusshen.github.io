路由

路由

路由可以通俗的理解为请求URL和控制器(Controller)的映射表，通过请求的URL可以触发某个控制器中的某个函数。

路由定义

路由的定义文件在项目目录src/{application}/app/router.js中，开发者可在该文件中进行映射关系的配置： 

module.exports = app => {
  router.verb(['router-name'], 'path-match', [middleware1, ..., middlewareN], app.controller.fileName.functionName);
};

其中[]内的参数代表可选项，下面针对该定义进行详细的介绍：

  • verb - 代表http的请求方式，如比较常用的get和post，同时也支持RESTful风格，RESTful方式具有更强的语义
    1. router.head - HEAD
    2. router.options - OPTIONS
    3. router.get - GET
    4. router.put - PUT
    5. router.post - POST
    6. router.patch - PATCH
    7. router.del - DELETE
    8. router.redirect - 重定向
  • router-name - 路由别名，可通过Helper辅助函数pathFor和urlFor反向生成对应的路由URL（可选项）
  • path-match - 路由地址
  • middleware1 - 可配置的中间件，可配置多个
  • app.controller.fileName.functionName - 所映射的控制器，其中app.controller是框架内部所提供的全局对象，fileName则是src/{application}/app/controller内的控制器文件名，functionName则是该文件内所要映射的函数名。一旦path-match被命中，则该function会被触发

常用API

获取URL上的get参数方式
// app/router.js
module.exports = app => {
  app.router.get('/search', app.controller.search.index);
};

// app/controller/search.js
exports.index = async ctx => {
  // search: helloWorld
  ctx.body = `search: ${ctx.query.name}`;
};

// curl http://127.0.0.1:7001/search?name=helloWorld
参数命名方式
// app/router.js
module.exports = app => {
  app.router.get('/user/:id/:name', app.controller.user.info);
};

// app/controller/user.js
exports.info = async ctx => {
  // user: 123, xiaoming
  ctx.body = `user: ${ctx.params.id}, ${ctx.params.name}`;
};

// curl http://127.0.0.1:7001/user/123/xiaoming
复杂参数的获取

路由里面也支持定义正则，可以更加灵活的获取参数： 

// app/router.js
module.exports = app => {
  app.router.get(/^\/package\/([\w-.]+\/[\w-.]+)$/, app.controller.package.detail);
};

// app/controller/package.js
exports.detail = async ctx => {
  // 如果请求 URL 被正则匹配， 可以按照捕获分组的顺序，从 ctx.params 中获取。
  // 按照下面的用户请求，`ctx.params[0]` 的 内容就是 `fastman2/1.0.0`
  ctx.body = `package:${ctx.params[0]}`;
};

// curl http://127.0.0.1:7001/package/fastman2/1.0.0
表单内容的获取
// app/router.js
module.exports = app => {
  app.router.post('/form', app.controller.form.post);
};

// app/controller/form.js
exports.post = async ctx => {
  // `{"name":"controller"}`
  ctx.body = `body: ${JSON.stringify(ctx.request.body)}`;
};

// 模拟发起 post 请求。
// curl -X POST http://127.0.0.1:7001/form --data '{"name":"controller"}' --header 'Content-Type:application/json'
